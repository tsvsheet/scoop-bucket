# scoop-bucket

The [Scoop](https://scoop.sh) bucket for the [tsvsheet](https://tsvsheet.com) CLIs (Windows):

```powershell
scoop bucket add tsvsheet https://github.com/tsvsheet/scoop-bucket
scoop install tsvsheet   # a spreadsheet for plain text
scoop install isnow      # match instants against isnow date/time patterns
```

Manifests in [bucket/](bucket/) are generated by [scripts/update.sh](scripts/update.sh) from the latest [tsvsheet.go](https://github.com/tsvsheet/tsvsheet.go) and [isnow.go](https://github.com/tsvsheet/isnow.go) releases — binaries built and attested by GitHub Actions in those repos, pinned here by sha256. A scheduled workflow keeps them current; never edit a manifest by hand. See [tsvsheet.com/install](https://tsvsheet.com/install/) for every install method.
